# KiloTrack 💪

A full-stack Progressive Web App (PWA) for tracking gym workouts and daily calorie intake, with AI-powered food and exercise suggestions.

## Features
- Log workouts with sets, reps, and weights
- Track daily calories and macros
- AI suggestions via Groq API (Llama 3)
- Firebase backend for auth and data sync
- Installable as a PWA on mobile

## Tech Stack
- HTML / CSS / JavaScript
- Firebase (Auth + Firestore)
- Groq API (Llama 3.3 70B)
- Service Worker for offline support

## Getting Started

```bash
git clone https://github.com/RiyanMujtaba/KiloTrack
```

Open index.html or deploy to Firebase Hosting. Add your Firebase config and Groq API key in the config file.
